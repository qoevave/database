Many 360¬∞ video players have limitations in either the maximum video resolution, or spatial audio compatability. The spatial alginment for the QoEVAVE database has been done for 360¬∞ Unity playback, with an external audio rendering engine.

warning

When using the separate B-Format ambisonics and 360¬∞ video files, be careful as to how these are combined together for audiovisual playback. Not all software will use the same (0¬∞,0¬∞) origin of the equirectangular video file. In such instances, an offset will occur that will rotate the video file causing misalignment with the audio.
